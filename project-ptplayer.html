PT Player

Platform Tennis is a fast growing sport on the coasts, and the client wanted to create a web application that tracked stats and head to head results. We designed and built this application from the ground up. It lets you see live scores for matches, livestream, and see head to head stats between any two teams. It was a fun site with a lot of interesting problems to solve.
